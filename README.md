# Voice Controlled Robot

A voice-controlled robot chassis using Arduino and Python. Say "forward", "back", "left", "right", or "stop" to control robot movement.

## Features
- Real-time voice recognition (0.5s response)
- Arduino + L298N motor driver
- Built-in laptop microphone support
- Clean terminal (no error spam)
- Simple 5-minute setup
